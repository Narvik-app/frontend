# Narvik - Front

Page du projet : https://docs.narvik.app/frontend/

## Fonctionnalités
- Interface moderne et responsive
- Tableau de bord avec statistiques avancées (Présences & Activités)
- Gestion complète des membres et des saisons
- Module de Point de Vente (POS) dédié
- Gestion des templates d'emails riches (TipTap)
- Filtres de recherche avancés pour les membres
- Support multi-rôles (Admin, Membre, Badgeuse)
